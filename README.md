## How to install
- Unity -> Window -> Package Manager  
- Click "+" at the top left corner  
- Add package from git URL  
- Insert `https://github.com/your-username/com.kiro.ide.integration.git`  
- Add  
- Done

## Features

- Auto-discover Kiro installations
- Generate .csproj files for IntelliSense
- Open files at specific line and column
- Option to reuse existing windows
- Support for C# 11.0

## Configuration

In **Edit → Preferences → External Tools**:

- **Reuse existing Kiro window**: Open files in existing window
- **Generate .csproj files for**: Choose which packages to generate for
- **Regenerate project files**: Regenerate project files

## Requirements

- Unity 2019.4+
- Kiro editor
